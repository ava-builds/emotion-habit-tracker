# User Flow

## Main User Journey

1. Open app
2. Daily emotion check-in
3. Select emotion
4. Adjust energy slider
5. Complete habit checklist
6. Optional short note
7. Save daily entry
8. View monthly emotional summary

---

## Main Screens

### Home Screen
- Daily overview
- Quick access to check-in
- Mood summary preview

### Daily Check-in Screen
- Emotion chip selector
- Energy slider
- Habit checklist
- Notes section

### Monthly Summary Screen
- Mood calendar
- Emotional heatmap
- Habit consistency visualization
- Insight cards

### Settings Screen
- Theme customization
- Habit management
- Notification settings
- Cycle tracking settings

---

## Core Experience Goals

- Fast daily interaction
- Emotionally calming interface
- Low-pressure tracking experience
- Minimal cognitive load
